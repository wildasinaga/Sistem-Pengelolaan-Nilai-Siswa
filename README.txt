|Sistem Pengelolaan Nilai Siswa|
 ------------------------------

This system was developed using ASP.NET MVC framework and localdb

The Function of this system are:
1. Add student
2. Add teacher
3. Add major
4. Add major for every teacher
5. Add student for every major
6. Export data into pdf

There are two roles in this system:
1. Admin
2. Teacher

Login:
Username and password that can be used for login s exist in Table Login.
